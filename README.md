# Selenium Page Object Model 
This repository demonstrates the implementation of the Page Object Model (POM) design pattern using Selenium WebDriver with Java. It showcases a clean and maintainable automation framework structure by separating test logic from page-specific elements and actions.

The project includes a sample implementation of POM, usage of Page Factory for element initialization, and method chaining to create readable and reusable test scripts.

This repository is useful for understanding how to design scalable Selenium automation frameworks following industry best practices.
